#HRNet application

## Getting Started

### Prerequisites
- Node.js 20 installed on your system.

### Installation
1. Clone the repository:
   ```bash
   git clone https://sebastiengallet.github.io/hrnet/

2. Navigate to the project directory
   ```bash
   cd hrnet

3. Install dependencies:
   ```bash
   npm install

4. Run for the development
   ```bash
   npm run dev

5. Build
   ```bash
   npm run build
